# Python Machine Learning Visualization Project




## Description

The CVE Visualizer is a machine learning and data visualization project that uses zero-day tracking datasets to model risk predictions. It references a zero-day dataset that categorizes common vulnerabilities & exposures based on operating system, exploit type, and software/hardware vendor for academic research.
